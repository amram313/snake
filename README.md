# 🐍 משחק הנחש

משחק Snake קלאסי בדפדפן, בנוי עם HTML, CSS ו-JavaScript טהור — ללא תלויות חיצוניות.

---

## תיאור

מימוש של משחק Snake הקלאסי בדפדפן. השחקן שולט בנחש, אוכל אוכל כדי לגדול ולצבור ניקוד, ומנסה להימנע מפגיעה בקירות או בגוף הנחש.

---

## פיצ'רים

- לוח משחק 20×20 עם רנדור על Canvas רספונסיבי
- ניקוד בזמן אמת + שיא ניקוד שנשמר ב-localStorage
- מערכת רמות — מהירות עולה כל 5 נקודות
- אוכל בונוס (כוכב זהב) — מופיע אקראית, שווה +3
- כפתורי **השהיה** ו-**התחל מחדש** תמיד גלויים
- מסך פתיחה ומסך Game Over
- אפקטים: particles, floating text, screen shake, glow
- שמע סינתטי (Web Audio API)
- שליטה: חצים, WASD, Space/P להשהיה, R להתחלה מחדש
- תמיכה במובייל: D-pad על המסך + swipe על ה-Canvas

|-----------|-------|
| HTML5 | מבנה הדף ו-Canvas |
| CSS3 | עיצוב, CSS Variables, Responsive |

---

## איך להריץ מקומית

1. הורד או שכפל את הריפו
2. פתח את `index.html` בדפדפן כלשהו
3. לחץ **"התחל משחק"** ותתחיל לשחק

לא נדרש שרת, לא נדרש npm, לא נדרש build.

---

## GitHub Pages

הפרויקט מוכן לפריסה ישירה דרך GitHub Pages:

1. עבור ל-**Settings → Pages** בריפו
2. בחר ב-Source: **Deploy from a branch**
3. בחר **main** ו-**/ (root)**
4. לחץ **Save** — האתר יהיה זמין בכתובת:
   `https://<username>.github.io/<repo-name>/`
