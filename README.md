# 🐍 משחק הנחש

משחק הנחש הקלאסי, בנוי עם HTML, CSS ו-JavaScript טהור — ללא תלויות חיצוניות.

---

## תיאור

מימוש של משחק Snake הקלאסי בדפדפן. השחקן שולט בנחש, אוכל אוכל כדי לגדול ולצבור ניקוד, ומנסה להימנע מפגיעה בקירות או בגוף הנחש עצמו.

---

## טכנולוגיות

| טכנולוגיה | שימוש |
|-----------|-------|
| HTML5 | מבנה הדף ו-Canvas |
| CSS3 | עיצוב, אנימציות וצבעים |
| JavaScript (Vanilla) | לוגיקת המשחק, רנדור, קלט |

---

## איך להריץ

1. הורד או שכפל את הפרויקט
2. פתח את הקובץ `snake.html` בדפדפן כלשהו
3. לחץ על **"התחל משחק"** ותתחיל לשחק

לא נדרש שרת, לא נדרש התקנה.
